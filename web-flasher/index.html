Azimuth

Flash Azimuth release firmware over USB — same look as the on-device portal.

Install firmware

Use a USB data cable. Works in Chrome or Edge on desktop (Web Serial). Connect your Seeed XIAO ESP32-C3, put it in bootloader if prompted (hold BOOT, tap RST, release BOOT).

Factory reset / clean install: When the installer asks, choose to erase flash first. That clears saved Wi‑Fi and portal settings (like “Erase saved settings” on the device). Then the image is written fresh.
After flashing

On your home Wi‑Fi, open the settings portal (mDNS):

http://azimuth.local:8080/

If .local does not resolve, use your router’s DHCP list for the board IP and port 8080. First-time setup can use the Azimuth-Setup AP at http://192.168.4.1/ (port 80).
