Put the latest firmware on your tracker from the browser.

Install

Use Chrome or Edge on a computer (not a phone). This browser can’t use USB serial, so flashing from here won’t work.
  1. Use a USB cable that carries data (not charge-only).
  2. Plug in a Seeed XIAO ESP32-C3.
  3. Click the button below and pick the serial port.
Stuck? Bootloader mode Hold BOOT, tap RST, release BOOT, then try the button again.

The progress dialog uses a generic time estimate; on this board, flashing is usually well under a minute.

Start fresh (optional)

If the installer asks whether to erase memory first, say yes if you want Wi‑Fi and saved settings cleared—the same idea as “erase saved settings” on the device.

When it’s done

  1. On your phone or computer, connect to the Azimuth-Setup Wi‑Fi network. It appears in your Wi‑Fi list after a first-time flash when you power the board on (before home Wi‑Fi is set up).
  2. A setup page should open, or go to 192.168.4.1 in a browser. Enter your home Wi‑Fi name and password, scroll to the bottom, and save so the board can join your network.
  3. On your home Wi‑Fi, on the same PC where you run OpenTrack, open azimuth.local:8080. Enter that PC’s network address in the UDP section, scroll down, and save—then use OpenTrack’s UDP input. More detail: quick start guide.

If azimuth.local doesn’t open, use the device’s IP from your router instead, still with :8080 at the end.
